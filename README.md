# the-opera
React based web apps playground

## To run it locally 


### for Sign in or login page 

      git clone [repo url]
      cd frontend 
      npm install
      npm run dev


### open new terminal and in that terminal 
      cd backend
      npm init -y
      node server.js

### create .env file and write 

JWT_SECRET = [your jwt token]

MONGO_URI = [you mongo bd url]

and you are good to go...
